## Scraping data from OSM or XML files

# Summary:
Goal is to extract data from a OSM and XML files
I adapted some codes from the Udacity course to store the data as pandas data frames and export them to csv files
Do kindly take note that data cleaning has not been performed

# scrape_xml.ipynb:
* Main code to extract data, store them as pandas dataframes and export them to csv files

# create_insert_into_sqlite.py:
* Create a SQLite database and tables
* Iterate and store values
* Insert values into database

# manipulate_data_example.py:
* Example code to extract data from the database

# singapore-sample.osm:
* OSM file used for this exercies
